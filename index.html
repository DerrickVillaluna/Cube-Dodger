Short Description

Grasshopper Adventure is a 2D game where players control a grasshopper avoiding sharp needles while navigating through a moving ground. The game challenges your reflexes and timing, providing an exciting and engaging gameplay experience.

Features

  • Unique up-and-down movement mechanics.
  • Challenging needle obstacles.
  • Dynamic moving ground for increased difficulty.
  • Scoring system to track player progress.

Development Tools

  • Unity Hub
  • C# Programming
  • Adobe Photoshop for design
